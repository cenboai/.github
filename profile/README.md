## Cenbo

北京森博明德营销科技股份有限公司（简称“森博营科”）是一家以AI驱动有效营销的科技公司。凭借20年在营销和视觉设计领域的深厚积累，公司已将丰富的行业数据和经验转化为自主研发的算法，并成功训练出AI视觉大模型。2023年，森博营科的自研大模型已正式对外发布并投入使用，为视觉内容生产力带来范式变革。

森博营科的业务覆盖多个领域，包括：
1. 营销科技：公司利用AI技术为营销领域提供创新解决方案，致力于提升营销效率和效果。
2. 高端品牌运营：专注于为高端品牌提供定制化运营服务，增强品牌影响力。
3. 顶级赛事及文旅IP运营：通过运营顶级赛事和文旅IP，拓展公司在文化、旅游和体育领域的业务版图。

2020年，森博营科成立了CeMeta森宇宙AI营销事业部，专注于AI营销大模型、AI设计工具和AI营销产品的产研开发及市场推广。该部门的成立标志着公司在AI营销领域的深入布局和专业发展。


随着人民对美好生活需求的不断增加，文旅行业在2023年迎来了复苏和振兴，展现出巨大的发展潜力。森博营科率先洞察到这一趋势，旗下其他业务板块已先行进入文旅产业进行探索，并取得了积极的推进效果。因此，公司的营销科技板块也在加速向文旅产业进军，积极寻求新的增长点和业务拓展。
森博营科希望通过不断的探索和创新，致力于为各行业提供有效、高效、可持续、可量化的营销服务和产品，积极推动产业的持续发展和升级。
